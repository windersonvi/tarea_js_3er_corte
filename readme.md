## Instalación de dependencias Backend
```bash
  cd server
  pip install -r requirements.txt
```

## Ejecutar server del Backend
```bash
  py app.py
```

## Instalación de dependencias Frontend
```bash
  cd taller_js
  npm install i
```

## Ejecutar server del Frontend
```bash
  npm run dev
```
